---------- Ajuda, Dicas e Procedimentos para Incluir um Nova Compra -------------

Cada compra só pode ter um único Fornecedor/Vendedor.
Caso o Fornecedor/Vendedor seja alterado, todos os produtos serão excluídos.

Passo a passo para inclusão de uma Nova Compra:
- Selecione Fornecedor/Vendedor
- Informe Data Compra e Tipo Pagamento
- Os campos Observação e Número nota fiscal são opcionais
- Selecione Marca e Produto
- Informe Lote, Data Validade, Quantidade(Qtd), Preço Compra,
Preço Venda e Desconto Máximo(Desc Max).
- Os campos Preço Compra e Preço Venda vêm com preços sugeridos no cadastro do produto
- Adicione o produto clicando no botão ADD+
- Repita este processo até o incluir todos os lotes de produto.
- Clique no botão Salvar pra finalizar a compra.
Obs: Os campos marcados com asterisco(*) são de preenchimento obrigatório
---------- Ajuda, Dicas e Procedimentos para Alterar uma Compra -------------

- Alguns campos não são passíveis de edição
- Altere os campos desejados
- Caso queira Excluir um lote já adicionado, clique no ícone da lixeira(Excluir)
- Clique no botão Salvar para que as alterações sejam confirmadas

- Caso queira cancelar a alteração, clique no botão Voltar
