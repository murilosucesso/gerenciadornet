---------------- Ajuda, Dicas e Procedimentos para Editar/Alterar/Excluir um Usuário -----------------

- Preencha os campos que serão incluídos/alterados
Nome: Nome completo do usuario
Login: Nome simples sem espaco que será utilizado pelo usuario para efetuar login
Perfil: Perfil do usuário. Define acessos ao usuário conforme o cadastro de acessos Default cadastrados
Senha: Cadastre uma senha default(ex: 123456) pra o usuário e peça que ele altere
Confirmacao de Senha: Preencha com o mesmo valor que a senha. (Evita erros de digitacao)
CPF: Informe o CPF do usuário
Numero RG: Numero de identidade do usuário
Data Nascimento: Selecione a data de nascimento do usuário clicando sobre o calendário

Obs: Cadastre o usuário uma ÚNICA vez. Não poderá existir mais de um usuário com o mesmo login.
Caso o usuário tenha esquecido a senha, o Administrador poderá efetuar a alteração
- Alguns campos não são passíveis de edição
- Os campos marcados com asterisco(*) são de preenchimento obrigatório
- Clique no botão Salvar para que a(s) inclusão/alteração seja confirmada

- Caso queira Excluir(Desabilitar) um usuário, clique no botão Excluir
- Obs: A ação de Excluir(Desabilitar) um usuário, não significa que este será
excluído fisicamente do banco de dados. Podendo ser recuperado posteriormente.
Sempre que um usuário for excluído/alterado, será gravado um histórico com os dados
do responsável pela exclusão/alteração. - Caso queira cancelar a alteração, clique no botão Voltar
