---------------------- Ajuda, Dicas e Procedimentos para Editar/Alterar/Excluir -------------------------

- Preencha os campos que serão alterados
- Alguns campos não são passíveis de edição - Os campos marcados com asterisco(*) são de preenchimento obrigatório
- Clique no botão Salvar para que as alterações sejam confirmadas

- Caso queira Excluir(Desabilitar) a entidade selecionada, clique no botão Excluir
- Obs: A ação de Excluir(Desabilitar) uma entidade, não significa que esta será
excluída fisicamente do banco de dados. Podendo ser recuperada posteriormente.
Sempre que uma entidade for excluída/alterada, será gravado um histórico com os dados
do responsável pela exclusão/alteração. - Caso queira cancelar a alteração, clique no botão Voltar
